# TONVERSE — Personal Site & Blog

Source for `tungedng2710.github.io`, a personal website and blog built with Jekyll (WhatATheme).

## Quick Start
- Prerequisites: Ruby and Bundler installed.
- Install: `bundle install`
- Serve locally: `bundle exec jekyll serve`
- Visit: http://localhost:4000

## Structure
- `_posts/` — blog posts (Markdown)
- `_includes/` — shared partials (head, navbar, footer, etc.)
- `_layouts/` — page layouts (default, post, blog, project)
- `assets/` — images, css, js, and misc assets
- `blog.md`, `project.md` — index pages for posts and projects
- `_config.yml` — site configuration (title, metadata, plugins)

## Notes
- Built for GitHub Pages; content is generated by Jekyll.
- Search uses Simple Jekyll Search via `search.json`.
